# The Melbourne Housing Market

An analysis on factors that influenced property prices in Melbourne, Australia between the years 2016-2018.

- Cleaned and Aggregated values using SQL

- Created visualisations using Vega-Lite.

- Developed website using HTML and CSS.

- Deployed to Github Pages.

Link to Visualisation: https://damian-ngau.github.io/Melbourne-Housing-Market/

[Dataset of Housing Market Data](https://www.kaggle.com/datasets/anthonypino/melbourne-housing-market)
